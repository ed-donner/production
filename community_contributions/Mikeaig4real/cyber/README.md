# Cybersecurity Analyzer

### Try the Live App: [Azure Mirror](https://cyber-analyzer.kindtree-adb49d11.southafricanorth.azurecontainerapps.io) or [GCP Mirror](TBD)

"Cybersecurity Analyzer" is a professional-grade code security auditing tool. It leverages high-performance AI agents and industry-standard security scanners to detect vulnerabilities, provide actionable remediation guidance, and calculate severity scores for Python applications.
