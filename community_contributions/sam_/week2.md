# Digital Twin

## Overview

This project is a digital twin application that represents a personalized AI assistant capable of interacting with users as the website owner. It is hosted using a GitHub Actions workflow to automate build and deployment to AWS, ensuring a seamless CI/CD pipeline.

## Features

* AI-powered conversational interface that mimics the persona of the site owner  
* Backend API built with FastAPI for handling AI requests
* Frontend built with Next.js for a modern, responsive UI 
* Automated CI/CD pipeline using GitHub Actions
* Deployment on AWS for scalability and reliability

---

## Tech Stack

* Frontend: Next.js (React)
* Backend: FastAPI (Python)
* AI Integration: Amazon Nova Lite  
* CI/CD: GitHub Actions
