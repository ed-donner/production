# AI-Powered SaaS Platform

## Overview
An AI-powered SaaS platform built to help users generate ideas and streamline professional workflows. It offers tools like a Business Idea Generator and AI-driven consultation summaries, making it easier to explore new opportunities and turn insights into action.




## Links

- Repo: https://github.com/Ngahu/ai-saas-web-app
- Live app: https://fgqwe34shn.eu-west-1.awsapprunner.com

## Features

### 1. Business Idea Generator
- Generate high-potential business concepts tailored for the modern AI economy.
- Ideas are formatted with headings, sub-headings, and bullet points for clarity.

### 2. Consultation Summaries
- Transform detailed meeting notes into professional, actionable summaries.
- Save time and improve client communication with AI-generated summaries.

### 3. Pricing Plans
- Flexible pricing plans to suit different user needs.
- Integrated with Clerk for user authentication and subscription management.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS
- **Backend**: FastAPI
- **Authentication**: Clerk
- **AI Integration**: OpenAI API
- **Environment Management**: Python dotenv
